# News Aggregator

## About

News Aggregator is a web application that collects news articles from various sources and presents them to users in a single platform. It aims to provide users with a convenient way to stay updated with the latest news across different topics and sources.

## Features

- **Aggregated News**: Fetches news articles from multiple sources and aggregates them in one place.
- **Categories**: News articles are categorized into different topics or sections for easy navigation.
- **Responsive Design**: User-friendly interface that adapts to different devices and screen sizes.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Sai6522/News-Aggregator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd News-Aggregator
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the server:

   ```bash
   node index.js
   ```

2. Open your web browser and visit [http://localhost:3000](http://localhost:3000) to access the News Aggregator application.

## Folder Structure

- `client`: Contains frontend code for the application.
- `constants`: Contains constants used throughout the application.
- `controller`: Contains controller logic for handling HTTP requests.
- `database`: Contains database configuration and schema files.
- `modal`: Contains database models for interacting with the database.
- `routes`: Contains route definitions for different API endpoints.
- `index.js`: Main entry point of the server.
- `package-lock.json` and `package.json`: Node.js package files.
